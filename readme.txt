1. Download & Extract Semua File Kedalam Folder DOTIndonesia Dan pindkan ke folder webserver (XAmpp/htdocs/)
2. Buat Sebuah Database Di PhpMyadmin dengan nama 'Lowongan'
3. Kemudian Import File Sql 'lowongan.sql' Kedalam database yang Sudah Di buat
4. aktifkan webserver(xampp) apache dan mysql
5. Buka Browser kemudian arahkan url ke folder webserver yang sudah ada folder DOTIndonesia (Localhost/DOTIndonesia)
